# PR Diff App
This app is a PR Diff reader

## What this app does
- Read-only access to GitHub pull requests
- Extract PR diffs and metadata
- No code modifications

## Permissions
- Pull requests: read-only
- Contents: read-only

## Data usage
- Optional storage for analysis
- No data is shared with third parties

## Security
- Uses GitHub App installation tokens
